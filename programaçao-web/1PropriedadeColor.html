Propriedade Color

A Propriedade (Color) pode ser inplementada de 3 formas, rgb, hexadecimal e cores pre-definidas Cores pre-definidas (ja vem prontas com seu codigo e nome ) e so selecionar

Forma prodefinida: .cor1{ color: blue; }

/*Cores rgb (uma mistura de 3 cores )*/

.cor2{ color: rgb(199, 71, 71); }

Cores hexadecimal representar Representada por um codigo

.cor3{ color: #FF1493;
